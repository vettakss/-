# Лабораторная работа №1

## Задание 1: Primes

Класс `Primes` находит и выводит все простые числа от 2 до 100.

**Как работает:**
- Метод `main` запускает цикл от 2 до 100
- Для каждого числа вызывает метод `IsPrime`
- Метод `IsPrime` проверяет, есть ли у числа делители от 2 до квадратного корня из этого числа
- Если делителей нет — число простое, метод возвращает `true`
- Если есть — возвращает `false`

## Задание 2: Palindrome

Класс `Palindrome` проверяет, является ли переданная строка палиндромом.

**Как работает:**
- Метод `main` получает строки из аргументов командной строки
- Для каждой строки вызывает метод `isPalindrome`
- Метод `reverseString` переворачивает строку с помощью цикла (идёт с конца в начало и собирает новую строку)
- Метод `isPalindrome` сравнивает исходную строку с перевёрнутой
- Если строки равны — это палиндром, если нет — не палиндром
