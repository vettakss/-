# Лабораторная работа №4

## Задание 1: Среднее арифметическое массива

Программа находит среднее арифметическое элементов массива с обработкой ошибок.

**Обрабатываемые исключения:**
- `ArrayIndexOutOfBoundsException` — выход за границы массива
- `NumberFormatException` — элемент не является числом
- `Exception` — все остальные ошибки

**Как работает:**
- Проверяет каждый элемент, является ли он числом (`instanceof Number`)
- Если нет — выбрасывает `NumberFormatException`
- Считает сумму и количество, вычисляет среднее

## Задание 2: Копирование файлов

Два варианта копирования содержимого `input.txt` в `output.txt`.

**Вариант 1 (ручное закрытие):**
- Использует `FileInputStream` и `FileOutputStream`
- Закрывает ресурсы в блоке `finally`
- Обрабатывает `FileNotFoundException` и `IOException`

**Вариант 2 (try-with-resources):**
- Ресурсы закрываются автоматически
- Более короткий и надёжный код

**Обрабатываемые ошибки:**
- Файл не найден
- Ошибки чтения/записи

## Задание 3: Собственное исключение

Программа выполняет математические операции с пользовательским исключением.

**Классы:**
- `CustomUnsupportedOperationException` — собственное исключение для неподдерживаемых операций
- `ExceptionLogger` — логирует исключения в файл `log.txt` с временем
- `MathOperations` — выполняет операции `+`, `-`, `*`, `/`
- `Main` — демонстрация работы

**Поддерживаемые операции:** `+`, `-`, `*`, `/`

**Если операция не поддерживается** — выбрасывается `CustomUnsupportedOperationException`

**Если деление на ноль** — выбрасывается `ArithmeticException`

**Логирование:** все исключения записываются в `log.txt`
