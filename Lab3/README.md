# Лабораторная работа №3

## Задание 1: Хеш-таблица (обобщенная)

Реализация хеш-таблицы с методом цепочек через связные списки.

**Класс HashTable<K, V>**

Методы:
- `put(key, value)` — добавляет пару, если ключ есть — обновляет значение
- `get(key)` — возвращает значение по ключу или null
- `remove(key)` — удаляет элемент по ключу
- `size()` — возвращает количество элементов
- `isEmpty()` — проверяет, пуста ли таблица
- `printTable()` — выводит содержимое

Принцип: хеш-функция вычисляет индекс (`hashCode() % размер массива`), по индексу хранится связный список. При коллизии элементы добавляются в тот же список.
