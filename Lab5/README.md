# Лабораторная работа №5: Регулярные выражения в Java

**NumberFinder** — в коде используется паттерн `\\d+(\\.\\d+)?`, который находит целые числа (одна и более цифр) и дробные числа (точка с последующими цифрами). Matcher перебирает все совпадения в тексте и выводит их.

**PasswordValidator** — регулярное выражение `^(?=.*[A-Z])(?=.*\\d)[A-Za-z0-9]{8,16}$` проверяет: позитивный просмотр вперёд требует наличия заглавной буквы и цифры, а затем проверяется, что весь пароль состоит только из латинских букв и цифр длиной от 8 до 16 символов.

**UpperAfterLower** — шаблон `([a-z][A-Z])` находит любую строчную букву, за которой сразу идёт заглавная. Метод `replaceAll("!$1!")` заменяет каждое найденное сочетание на то же самое, но обрамлённое восклицательными знаками.

**IPValidator** — сложное регулярное выражение проверяет каждый из четырёх октетов: числа от 0 до 255 (25[0-5], 2[0-4]\d, 1\d{2}, [1-9]?\d). Точки между октетами проверяются через `\.{3}` для первых трёх.

**WordFinder** — программа принимает заданную букву, преобразует её в нижний регистр и строит паттерн `буква + \\w*` с флагом `CASE_INSENSITIVE`, что позволяет находить слова, начинающиеся с этой буквы в любом регистре.
